## Usage

1. Double-click CANDataConverter.exe to run
2. Select your CAN database file (.dbc)
3. Select your CAN log file (.blf or .asc)
4. Choose output format and time axis option
5. Click "Convert!"

## System Requirements

- Windows 10/11 (64-bit)
- No Python installation required
